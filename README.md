# 모두의 성적 데이터 해석 가상 프로그램

연수용(학생/학부모)으로 배포할 수 있는 **가상 시뮬레이션 웹 페이지**입니다.  
GitHub Pages에 그대로 올리면 바로 동작합니다.

## 포함 파일
- `index.html`
- `style.css`
- `script.js`
- `samples.json` (가상 학생 예시 10개)

## GitHub Pages로 배포 방법(가장 쉬운 방식)
1) GitHub에서 새 저장소(repository) 생성  
2) 이 폴더의 파일 4개를 저장소 루트에 업로드  
3) GitHub 저장소 → **Settings** → **Pages**  
4) **Branch = main**, **/(root)** 선택 → Save  
5) 생성된 URL로 접속하면 실행됩니다.

## 주의
- 본 페이지는 “입학사정관이 이런 식으로 해석하지 않을까?”라는 관점의 **연수용 가상 시뮬레이션**입니다.
- 실제 대학의 평가 결과/기준을 단정하지 않습니다.
- 샘플 데이터는 모두 익명 가상 예시입니다.

(원하시면) 실제 학교/과목 데이터를 넣는 버전으로도 확장 가능합니다.
